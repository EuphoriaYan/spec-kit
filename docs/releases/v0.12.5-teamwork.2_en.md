# v0.12.5+teamwork.2

[中文主文档](v0.12.5-teamwork.2.md)

`v0.12.5+teamwork.2` is the second pinned AI Team distribution based on
upstream Spec Kit `v0.12.5`, and the first fixed release containing all six
primary Team Skills and the advanced Memory extension entry.

## Distribution Scope

- default Team-minimal installation for Codex, Claude Code, Cursor, and Trae;
- six focused Team Skills: Specify, Plan-and-Task, Assess, Fix, Implement, and
  Review;
- an explicit Memory Consolidate advanced entry for post-delivery lessons and
  Knowledge promotion, outside automatic Feature/Bugfix routing;
- separate Feature and Bugfix paths with a shared Review/Assess/Fix quality
  loop;
- direct CodeGraph impact analysis during planning and defect assessment;
- local-only Feature and Bugfix work packages;
- Chinese primary human guides with `_en` English backups.

## Fixed Versions

- Git tag: `v0.12.5+teamwork.2`
- Python package version: `0.12.5+teamwork.2`
- Upstream base: `github/spec-kit@v0.12.5`
- CodeGraph CLI: `>=1.0.0,<2.0.0`
- Python: `>=3.11`

## Install

```bash
uv tool install specify-cli --force \
  --from git+https://github.com/EuphoriaYan/spec-kit.git@v0.12.5+teamwork.2
```

Then initialize the real coding repository:

```bash
specify init . --integration <codex|claude|cursor-agent|trae>
```

When upgrading from `v0.12.5+teamwork.1` or a temporary `main` installation,
rerun installation and initialization, then inspect the six primary Skills,
advanced extension entry, managed AI rule, `.gitignore`, and project-owned Team
configuration. To roll back,
reinstall the previous reviewed tag and inspect initialization changes before
overwriting project files.
