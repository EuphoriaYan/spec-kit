# v0.12.5+teamwork.3

[English backup](v0.12.5-teamwork.3_en.md)

`v0.12.5+teamwork.3` 继续绑定上游 Spec Kit `v0.12.5`，Team 扩展版本升级为
`0.7.0`。

## 主要变化

- Gate 提供中文含义和“下一步”等自然表达；
- Plan 区分业务软件、框架和外部前置需求，并检查同 PR/关联 PR 边界；
- 教程步骤使用确定性/可评价定义和四态证据；
- Team profile 初始化前强制检查 CodeGraph 1.x，缺失时安全失败并提供安装指引；
- 精确忽略本地索引、工作包和生成的 Team Skills；
- GitCode 使用能力探测、写后回读和安全降级。

## 发布与安装

合并发行 PR 后，在合并提交创建 `v0.12.5+teamwork.3` tag。tag 存在并完成四种
integration 冒烟验证后，使用：

```bash
uv tool install specify-cli --force \
  --from git+https://github.com/EuphoriaYan/spec-kit.git@v0.12.5+teamwork.3
specify init . --integration <codex|claude|cursor-agent|trae>
```

需要回滚时重新安装 `v0.12.5+teamwork.2`，再执行相同的项目刷新命令。
