# v0.12.5+teamwork.3

[中文主文档](v0.12.5-teamwork.3.md)

`v0.12.5+teamwork.3` remains based on upstream Spec Kit `v0.12.5` and updates
the Team extension to `0.7.0`.

## Highlights

- localized gates accept natural replies such as “下一步”;
- Plan classifies business, framework, and external prerequisites and checks
  combined-PR versus linked-PR delivery;
- tutorial steps use deterministic/evaluable definitions and four-state evidence;
- Team profile initialization requires CodeGraph 1.x and fails safely with
  installation guidance when it is unavailable;
- local indexes, work packages, and generated Team Skills are ignored precisely;
- GitCode uses capability probing, write read-back, and a safe fallback.

## Release and install

After merging the release PR, create `v0.12.5+teamwork.3` on the merge commit.
Once the tag exists and all four integrations pass smoke tests, install with:

```bash
uv tool install specify-cli --force \
  --from git+https://github.com/EuphoriaYan/spec-kit.git@v0.12.5+teamwork.3
specify init . --integration <codex|claude|cursor-agent|trae>
```

To roll back, reinstall `v0.12.5+teamwork.2` and run the same project refresh.
