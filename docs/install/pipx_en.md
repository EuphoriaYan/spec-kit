# Install with pipx

[中文主文档](pipx.md)

uv is the default. In a pipx-standardized environment:

```bash
pipx install --force git+https://github.com/EuphoriaYan/spec-kit.git@v0.12.5+teamwork.3
specify --version
```

Then follow the [installation guide](../installation_en.md). Keep the Team on
the same reviewed tag.
